Quick Start

Embedding

  1. Select the image you want to embed data in.
  2. Add the data to be embedded.
  3. Enter a password for data encryption.

Extraction

  1. Select the image you want to extract data from.
  2. Enter the password used for data encryption.

Tips & Tricks

  • Larger images can hold more data.
  • Lossless images (e.g. .png) can hold more data.
